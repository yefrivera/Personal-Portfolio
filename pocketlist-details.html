Descripción general

Aplicación móvil multiplataforma orientada a la gestión personalizada de listas de compras. Fue implementada utilizando Flutter como framework de desarrollo y Firebase como plataforma backend, lo que permitió integrar funcionalidades de autenticación, almacenamiento de datos y sincronización en tiempo real. La aplicación busca facilitar al usuario el seguimiento y control de los productos que necesita adquirir, integrando una interfaz sencilla e intuitiva que promueve una experiencia de usuario ágil y eficiente.

Entre las funcionalidades principales se encuentra la posibilidad de crear listas de compras con múltiples productos, asociando cada producto a un sitio específico donde se desea adquirirlo. Además, la aplicación permite registrar nuevos sitios de compra mediante formularios modales. Los productos pueden marcarse como comprados mediante gestos táctiles (swipe o doble tap), modificarse o eliminarse siempre que no estén marcados. La base de datos en Firebase almacena toda esta información, estructurada en colecciones que representan listas, productos y sitios, con sus respectivos identificadores y fechas de registro.

También permite crear nuevas listas a partir de otras ya existentes mediante clonación, lo cual facilita la reutilización de compras frecuentes. Esta funcionalidad se presenta de forma accesible y clara desde la interfaz principal.

Finalmente, La aplicación incorpora una autenticación mediante login y contraseña, donde las credenciales del usuario se validan y almacenan de forma segura, utilizando cifrado RSA de 256 bits para las contraseñas para mayor seguridad de los datos de acceso de usuario.

La applicación se desplegó en la web a manera de prueba. Sin embargo, su uso más optimo se lleva a cabo en dispositivos móviles.
