Aplicativo Web de Realidad Virtual

  1. Inicio
  2. Aplicativo Web de Realidad Virtual

Descripción general

Aplicativo web inmersivo desarrollado con tecnologías WebXR y Three.js, que permite a los usuarios realizar un recorrido virtual interactivo por los cuatro destinos más representativos del departamento del Cauca, Colombia: Popayán, Silvia, Puracé y Tierradentro. A través de una experiencia accesible desde navegadores compatibles con realidad virtual, el usuario puede explorar escenarios en 360 grados, enriquecidos con contenido multimedia y elementos interactivos que fomentan la promoción turística y cultural de la región. El sistema busca ofrecer una vivencia envolvente y accesible sin necesidad de instalaciones adicionales, aprovechando estándares abiertos de la web para la realidad virtual.

Tecnologías utilizadas
  • WebXR: Permite la implementación de experiencias inmersivas directamente desde el navegador web, sin necesidad de aplicaciones externas ni instalaciones adicionales.
  • Three.js: Biblioteca JavaScript para gráficos 3D, utilizada para renderizar escenas tridimensionales de alta calidad y manipular elementos interactivos dentro del entorno virtual.
  • HTML5, CSS y JavaScript: Para la estructura, estilos y lógica de la aplicación web.
  • Contenido multimedia: Fotografías y videos en 360°, así como recursos sonoros y visuales que complementan la inmersión.
Características destacadas
  • Compatible con gafas de realidad virtual como las Oculus Quest 2 y 3.
  • Navegación libre por los distintos destinos turísticos.
  • Integración de hotspots interactivos con contenido informativo y multimedia.
  • Carga eficiente de recursos optimizada para diferentes velocidades de red.
  • Diseño modular que permite escalar fácilmente el proyecto e incorporar nuevos destinos o funcionalidades.

El proyecto representa una aplicación concreta de la ingeniería en tecnologías inmersivas, combinando conocimientos en desarrollo web, modelado 3D, experiencia de usuario (UX) y turismo digital.
